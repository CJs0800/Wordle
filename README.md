This project was developed in 2024 as part of a group university project under a (very) limited time constraint.  
The objective was to deliver a functional and user-friendly solution as a team while applying the skills acquired in UI/UX design and Frontend development.

---

## Features

- Wordle-style gameplay 
- Color feedback system (correct / present / absent letters)
- Interactive on-screen keyboard
- Word validation and guess checking
- Dynamic grid interface
- Built with Vue 3 and Vite

---

## Supported Platforms

-   ✅ Web ([ GitHub Pages ](https://CJs0800.github.io/Wordle))
